Logika Backend Test
Descripción

API desarrollada en FastAPI para gestionar usuarios y tareas (CRUD). Incluye autenticación con JWT y manejo de base de datos con PostgreSQL. Se usan migraciones con Alembic y seguridad de contraseñas con bcrypt.

Tecnologías

Python 3.11

FastAPI

SQLAlchemy

PostgreSQL

Alembic (migraciones)

Passlib (hash de contraseñas)

JWT (auth)

Uvicorn (servidor ASGI)

PgAdmin 4 (opcional, para visualizar la base de datos)

Requisitos

Python 3.11.8

PostgreSQL

Virtual environment (venv)
